# 🛒📦 *# Istok!* 1.0.2
###  -> Seu sistema de controle de estoque.

<br/>
Projeto desenvolvido em **🐍Python** com o objetivo de simular um sistema simples de controle de estoque para mercado, permitindo cadastro, listagem e cálculo de itens armazenados.

O programa é executado via terminal e utiliza estruturas fundamentais da linguagem, como:

- Listas
- Dicionários
- Funções
- Estruturas de repetição
- Validação de dados
- Tratamento de exceções
- _match-case_
<br/>


 ## Objetivo do projeto. 🎯

<br/>
Desenvolver um sistema simples aplicando os conhecimentos adquiridos, com foco em:

- Organização de dados em memória
- Validação de entradas do usuário
- Prevenção de duplicidade de informações
- Aplicação de boas práticas de programação
<br/>

 ## Funcionalidades. ⚙️
<br/>
No nosso Menu Interativo, onde é utilizado match-case para navegação, você pode:<br/>
<br/>
1 — Exibir itens em estoque<br/>
<br/>Mostra todas as informações dos produtos cadastrados com:

- Nome
- Código de barras
- Valor (formatado parámoeda R$)
- Quantidade disponível
<br/>
<br/>2 — Cadastrar novo produto<br/>
<br/>Solicitando as informações do produto para o usuário, como nome, código de barras, valor e quantidade a serem adicionados.<br/>

<br/>Possuindo:

- Tratamento de erro para entradas inválidas
- Validação para que o código de barras não seja utilizado em mais de um produto
- Validação para que os Valores e Quantidades não sejam negativos.
- Formatação automática<br/>

<br/>3 — Exibir total de itens no estoque<br/>
<br/>Percorre a lista de produtos e soma todas as quantidades armazenadas em seu estoque.
